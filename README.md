# qiyamah-site

Признаки Судного дня — статический сайт с 52 признаками.

## Локальная разработка

```bash
npm install
npm run dev
```

Сайт будет доступен на http://localhost:5173

## Сборка и деплой

Проект автоматически собирается и деплоится на GitHub Pages при пуше в `main` (workflow `.github/workflows/deploy.yml`).

Локальная сборка:

```bash
npm run build
npm run preview
```
